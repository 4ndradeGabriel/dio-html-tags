O que é HTML e como surgiu?

HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar páginas na web. Surgiu em 1991, desenvolvido por Tim Berners-Lee, para estruturar documentos na internet através de hipertextos, links que conectam diferentes páginas. HTML é essencialmente o esqueleto de uma página web, definindo sua estrutura e conteúdo.

Tags para Texto

HTML possui várias tags para a formatação de texto. Algumas das mais comuns são:

  • <h1> a <h6>: Usadas para títulos, onde <h1> é o título principal e <h6> o menos importante.
  • <p>: Define um parágrafo.
  • <strong> e <em>: Respectivamente, deixam o texto em negrito e em itálico.
  • <blockquote>: Utilizado para citar blocos de texto de outras fontes.
  • <ul>, <ol> e <li>: Usadas para criar listas, sendo <ul> para listas não ordenadas (com marcadores) e <ol> para listas ordenadas (numeradas). Os itens da lista são definidos com <li>.
  • <br>: Insere uma quebra de linha dentro de um parágrafo ou texto.
  • <pre>: Mantém a formatação de texto e espaços em branco, exibindo o conteúdo exatamente como está no código fonte.
  • <code>: Utilizado para exibir trechos de código em uma fonte monoespaçada.

Tags para Multimídia

HTML também permite a inserção de conteúdo multimídia como imagens, vídeos e áudios:

  • <img>: Insere imagens na página. Atributos importantes incluem src (fonte da imagem) e alt (texto alternativo).
  • <video>: Insere vídeos, podendo incluir controles como play, pause e volume. É possível especificar múltiplas fontes para compatibilidade com diferentes navegadores.
  • <audio>: Insere áudios com opções de controle semelhantes ao vídeo, e também pode ter múltiplas fontes.
  • <iframe>: Permite a inserção de um conteúdo de outra página da web, como vídeos do YouTube ou mapas do Google.

Seletores HTML

Seletores HTML ajudam a identificar e manipular elementos específicos na página. Alguns exemplos:

  • ID: Identifica um único elemento com o atributo id, usando <div id="exemplo">. IDs devem ser únicos em uma página.
  • Classes: Identificam múltiplos elementos com o atributo class, usando <div class="exemplo">. Classes podem ser reutilizadas em vários elementos.
  • Elementos: Podem ser selecionados diretamente pelo nome da tag, como <p> para parágrafos.

Tags Semânticas

Tags semânticas em HTML ajudam a dar significado ao conteúdo, facilitando a compreensão para desenvolvedores e máquinas:

  • <header>: Define o cabeçalho da página ou de uma seção, geralmente incluindo o logotipo, título, e navegação principal.
  • <nav>: Especifica uma seção de links de navegação.
  • <article>: Indica um conteúdo independente e auto-suficiente, como uma postagem de blog ou uma notícia.
  • <section>: Define uma seção genérica dentro de um documento, geralmente com um tema relacionado.
  • <aside>: Representa um conteúdo lateral, como uma barra lateral ou caixas de informação.
  • <footer>: Representa o rodapé do documento ou de uma seção, geralmente contendo informações sobre o autor, links para documentos relacionados, e a declaração de direitos autorais.
  • <main>: Delimita o conteúdo principal da página, excluindo conteúdos repetidos em múltiplas páginas, como barras laterais, rodapés e cabeçalhos.

Formulários em HTML

Formulários são fundamentais para a interação do usuário com o site. As principais tags incluem:

  • <form>: Contém todos os elementos de um formulário e define como os dados são enviados.
  • <input>: Utilizado para campos de entrada de dados, com diversos tipos, como texto, senha, email, número, entre outros.
  • <textarea>: Define uma área de texto para entradas longas.
  • <button>: Cria um botão que pode ser usado para enviar o formulário ou realizar ações personalizadas.
  • <label>: Associa um rótulo a um elemento de formulário, melhorando a acessibilidade.
  • <select> e <option>: Criam listas suspensas para seleção de uma ou mais opções.

Links e Navegação

Os links são a base da navegação na web. As tags mais comuns incluem:

  • <a>: Define um link. O atributo href especifica o destino, que pode ser uma URL, um link para um arquivo ou um link âncora dentro da mesma página.
  • <nav>: Usada para agrupar um conjunto de links de navegação.

Tabelas em HTML

Tabelas são usadas para organizar dados em linhas e colunas. As principais tags são:

  • <table>: Contém toda a tabela.
  • <tr>: Define uma linha na tabela.
  • <td>: Define uma célula em uma linha, ou seja, o conteúdo da tabela.
  • <th>: Define um cabeçalho de uma coluna ou linha, destacando-o do resto.
  • <thead>: Agrupa os cabeçalhos da tabela.
  • <tbody>: Agrupa o corpo principal da tabela.
  • <tfoot>: Agrupa o rodapé da tabela, geralmente usado para somas ou resumos.

Elementos Interativos

HTML também suporta elementos interativos que podem melhorar a experiência do usuário:

  • <details>: Cria um widget que o usuário pode abrir ou fechar para revelar informações adicionais.
  • <summary>: Define um resumo ou legenda para o elemento <details>.
  • <dialog>: Cria uma janela de diálogo interativa que pode ser usada para mensagens, formulários ou interações com o usuário.
  • <progress>: Mostra o progresso de uma tarefa, como um carregamento de arquivo.
  • <meter>: Representa uma medida em um intervalo conhecido, como um indicador de volume.

Conclusão

HTML é a espinha dorsal de qualquer página web. Compreender as diferentes tags e como usá-las permite criar estruturas web ricas e bem organizadas. À medida que a web evolui, novas tags e funcionalidades são adicionadas, mantendo o HTML uma ferramenta essencial para desenvolvedores.
